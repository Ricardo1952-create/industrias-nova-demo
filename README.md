# Industrias Nova — Demo de cotización con registro visual

La demo mantiene un único recorrido rígido:

1. El visitante escribe `cotizar`.
2. Completa empresa, CUIT, nombre, WhatsApp y trabajo a cotizar.
3. Aparece la confirmación.
4. Se muestra cómo quedó registrada la consulta usando los mismos datos ingresados.
5. Finalmente se explican las funciones adicionales que podría tener el agente real.

El registro es visual y demostrativo; todavía no se guarda en una planilla real.
